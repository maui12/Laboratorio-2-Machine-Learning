## Instrucciones

Antes, se debe descargar el dataset de UTKFace, asegurandose que se encuentre en la carpeta dataset/UTKFace (o cambiando la ruta deseada en el archivo src/config.py)

1. **Abrir terminal en la raiz del proyecto, ejecutar:**
```
conda env create -f environment.yaml
```

2. **Activar entorno**
```
conda activate lab2ml
```

3. **Ejecutar main.py**
```
python main.py
```
